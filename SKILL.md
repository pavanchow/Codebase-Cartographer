**Key reminders for this brand:**
- Light-mode only, crisp white canvas with `#F9FAFB` panels.
- Accent is cobalt-indigo `#3B4BDB`; use sparingly.
- Inter for UI, JetBrains Mono for code / paths / counts. Mono is load-bearing.
- Lines, not shadows. 1-px borders do the work of hierarchy.
- Corner radii stay subtle (4 / 6 px). No pill-shaped cards, no gradients, no emoji.
- Voice: declarative, precise, no exclamation marks, sentence case.
- Icons: Lucide CDN only.
